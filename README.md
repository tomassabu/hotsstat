# hotsstat

Bruker HotsApi - https://hotsapi.net/docs
legg inn navn og battletag på spiller og generer en jsonfil med data om:
  player,
  hero,
  winner,
  hero dmg,
  siege dmg,
  exp contribution,
  time spent dead,
  data of game
